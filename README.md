# Python Data Analysis Practice 2

Python data analysis practice using **pandas** and **matplotlib**.  
This repository focuses on working with CSV data and performing basic data analysis tasks.

Includes:
- CSV loading
- Data inspection
- Handling missing values
- Data filtering
- Basic data visualization

## Dataset

The dataset (`students.csv`) contains fictional student performance data with the following fields:
- student_id
- name
- subject
- quiz_score
- exam_score
- attendance

Some values are intentionally missing to practice data cleaning.

## How to Run

1. Make sure you have Python installed (preferably 3.x)
2. Install required libraries:
   ```bash
   pip install pandas matplotlib
   ```
3. Make sure students.csv is in the same folder as the script
4. Run the Script
   ```bash
   python analysis.py
   ```
